# Menu 导航菜单

导航菜单一般用于在网站内的跳转，一般分为**顶部导航**和**侧边导航**

## 模式选择

Menu 组件有水平、垂直两种模式选择，水平模式一般用于顶部导航，而垂直模式一般用于侧边导航

  <div>

      <MenuItem>视频</MenuItem>

    </Menu>

</Preview>

## 内嵌菜单

内嵌菜单在水平和垂直模式的展现会有所不同，水平菜单中会以弹出的形式出现，而垂直菜单则会以展开的形式出现。同时，内嵌菜单支持多级内嵌。
