# Button 按钮

按钮用于触发用户指定的一组行为

## 类型

`Wing`为用户提供了多种类型的按钮类型

- Primary 主按钮：一个区域内一般只需要一个
- Default 默认按钮：无法分清主次的按钮
- Danger 危险按钮：用于进行危险操作，如删除等等
- Link 链接按钮：用于导向外部链接
